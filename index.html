05 — Keeping a safe copy

This ledger saves itself automatically in this browser, on this device, every time something changes — no internet needed. But if this browser's data is ever cleared, or you open the file on a different device, that saved copy won't follow you. Download a backup now and then, and keep the file somewhere safe (email it to yourself, save it to a USB stick, etc.).
